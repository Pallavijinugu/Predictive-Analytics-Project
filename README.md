# 📊 Predictive Analytics Using Historical Data (Sales Forecasting)

## 📌 Project Overview
This project builds a machine learning model to analyze historical sales data and predict future sales trends using regression techniques. It helps understand how data-driven forecasting works in real-world business scenarios.

---

## 🎯 Objective
- Analyze historical sales data
- Build a predictive model using Linear Regression
- Forecast future sales trends
- Evaluate model performance
- Visualize insights using graphs

---

## 📂 Dataset
**Source:** Sample Superstore Dataset  
**Columns used:**
- Order Date
- Sales

Optional original dataset also contains:
- Category
- Sub-Category
- Region
- Profit
- Quantity

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Machine Learning Model
- Linear Regression

---

## 🧠 Workflow

### 1. Data Collection
Load dataset from CSV file.

### 2. Data Cleaning
- Convert Order Date to datetime
- Handle missing values
- Select required columns

### 3. Feature Engineering
- Convert dates into numerical format (Days)

### 4. Model Building
- Split dataset into training and testing sets
- Train Linear Regression model

### 5. Prediction
- Predict sales on test data
- Forecast future sales trends

### 6. Evaluation
- Mean Absolute Error (MAE)
- R² Score
